## 🎯 Contexte
Analyse des avis clients d'une entreprise de grande distribution (ou d'un club de football). L'objectif est d'améliorer les compétences en requêtage SQL avancé tout en respectant le RGPD.

## 📚 Compétences développées
- Créer une base de données conforme aux normes réglementaires et aux besoins clients
- Effectuer des requêtes SQL complexes pour répondre à une problématique métier
- Gérer une base de données en respectant le RGPD

## 🧠 Ce que j'ai appris
Ce projet a consolidé mes compétences SQL avec des requêtes plus avancées (sous-requêtes, vues, procédures) et m'a appris à structurer une base de données autour d'un besoin métier précis tout en garantissant la conformité réglementaire.

## 🗂️ Livrables
- Modèle de base de données (MCD/MLD)
- Scripts SQL (création, insertion, requêtes)
- Rapport d'analyse de la satisfaction client

## 🔧 Outils utilisés
SQL , EXCEL
